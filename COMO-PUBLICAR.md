# Como publicar o Meu Bolso e instalar no celular

## Passo 1 — Criar conta no GitHub (gratuito)
1. Acesse https://github.com/signup
2. Crie uma conta gratuita (email + senha)

## Passo 2 — Criar repositório
1. Clique no botão verde **"New"** ou acesse https://github.com/new
2. Nome do repositório: `meubolso` (tudo minúsculo, sem espaços)
3. Deixe marcado como **Public**
4. Clique em **"Create repository"**

## Passo 3 — Fazer upload dos arquivos
1. Na página do repositório, clique em **"uploading an existing file"**
2. Arraste TODOS os arquivos desta pasta:
   - `index.html`
   - `manifest.json`
   - `sw.js`
   - A pasta `icons/` com os dois ícones dentro
3. Clique em **"Commit changes"**

## Passo 4 — Ativar GitHub Pages
1. No repositório, clique em **Settings** (aba no topo)
2. No menu lateral esquerdo, clique em **Pages**
3. Em "Branch", selecione **main** e clique em **Save**
4. Aguarde ~2 minutos e atualize a página
5. Você verá a URL do seu app: `https://SEU-USUARIO.github.io/meubolso`

## Passo 5 — Instalar no celular

### Android (Chrome)
1. Abra a URL do app no Chrome
2. Um banner **"Adicionar à tela inicial"** vai aparecer automaticamente
3. Ou: toque nos 3 pontinhos (...) → **"Adicionar à tela inicial"**
4. O ícone do Meu Bolso aparece na sua tela inicial!

### iPhone (Safari)
1. Abra a URL do app no Safari (não funciona no Chrome do iPhone)
2. Toque no botão de **compartilhar** (ícone de seta para cima)
3. Role e toque em **"Adicionar à Tela de Início"**
4. Toque em **Adicionar**
5. O ícone aparece na sua tela inicial!

## Dados salvos
- Todos os dados ficam no armazenamento local do seu celular/computador (localStorage)
- O app funciona offline após a primeira visita
- Para fazer backup: no futuro você pode exportar os dados pela tela de Configurações

## Acessar em vários dispositivos
- Abra a mesma URL em cada dispositivo e instale em cada um separadamente
- Os dados são locais por dispositivo (não sincronizam automaticamente entre celular e PC)
